# Radius

a custom theme for rockbox devices of 320 x 240 resolution. 
there are only two screens here : sbs and wps which will handle the base screens (folder navigation, settings etc) and the now playing screen respectively.

## current fonts
- Gabarito : semibold / bold
- Inter tight : content text
- MicroNotoSans for CJK support (combined into the two above)
